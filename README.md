# 📝 noco-form: 生产级开源表单编辑器



  
#### **快速开始**：
  
```SHELL
git clone git@github.com:bojue/noco-form.git 

cd noco-form  

yarn 

npm run dev
```

#### 特点

- **SaaS 核心能力**：基于 Vue 3.5 + TS，内置 57 种业务组件（含电子签名/NPS）与 31 项精细配置模块。

- **极致交互体验**：金融级拖拽引擎，支持 1:1 响应式预览，确保 50+ 复杂字段操作流畅不掉帧。

- **工业级架构**：编辑器与渲染器深度解耦，内置 Design Tokens、中英 i18n 与深色模式，支持分钟级无缝嵌入。

- **高性能基座**：采用 Vite 5.4 + Pinia 3.0 + Tailwind CSS 3.4 现代技术栈，代码 100% MIT 开源。
